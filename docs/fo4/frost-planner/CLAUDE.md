## Environment

- Running in WSL/Git Bash on Windows
- Use Unix commands, not Windows commands:
	- Use `ls` instead of `dir`
	- Use `find` or `ls -R` instead of `dir /s`
	- Use `cat` instead of `type`
	- Use `rm` instead of `del`
	- Use forward slashes `/` in paths, not backslashes `\`
- Do not use commands that would require manual permissions.
